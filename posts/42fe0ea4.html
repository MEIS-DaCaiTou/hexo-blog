搭建永久免费梯子！

发表于 2025-02-22|更新于2025-02-22|技术教程
|字数总计:2342|阅读时长:7分钟|阅读量:

搭建永久免费梯子！Cloudflare赛博大善人的终极使用指南

前言

Cloudflare在12月份发布了协议，规定不可以使用Cloudflare Workers & Pages搭建代理服务，否则可能会导致封禁账号。如果你害怕被封号，可以使用虚拟邮箱来创建一个Cloudflare小号，并将域名托管至小号。

众所周知，旁白博主在几万年之前发布了一篇博文——自动获取Greenhub激活码：源码分析与脚本编写，由此也造福了很多朋友们。

好消息是，在不久之前，他发现了这样一个项目：https://github.com/cmliu/edgetunnel。这位大佬的项目，可以通过 Cloudflare 中的 Workers 和 Pages 来搭建永久免费的佛跳墙，且速度飞快！这也是我放弃了Greenhub的原因之一。

需要用到的东西有：

  • 一个电子邮件（这个应该都要有吧）
  • 一个Cloudflare账号
  • 一个域名（免费）
  • 佛跳墙使用工具（例如v2ray，clash等）

注册一个Cloudflare账号

点击这里，你会进入以下界面：

如果你可以使用Google账号或者Apple ID注册，也行。如果没有，就输入自己的邮箱与密码，点一下注册就好了，如果像图中那样有人机验证的，验证一下即可。如果有要求验证电子邮件，那么打开你的电子邮件，点一下Cloudflare的验证链接即可。

创建Cloudflare Workers

注册完毕后，点一下左上角的Cloudflare图标，就可以来到主界面，然后你应该会看见这样的界面：

（如果没有中文，可以点击右边English按钮，切换至简体中文或繁体中文）

我们根据图中的标注，进入 Workers 和 Pages 页面。然后点击右上方的 创建 按钮，点击创建Worker或者快速开始，随后点击部署，名称不要改。如果想改，不要取名叫VPN、Shadowshare等名字，否则给屏蔽了就倒霉了。

部署完后，点击编辑代码，将默认代码全删。然后进入这里复制代码（点击右上角Raw旁边的复制按钮），将代码复制（代码已加密，且自带优选IP的功能）。若打不开该网站的，可以使用上一篇文章所讲的Greenhub暂替。

然后将代码粘贴至刚才删除代码的地方，再点击右上角的部署，等按钮变灰，就是部署完成。

现在再来设置密码，点击左上角返回，再点击顶部的设置，接着找到变量和机密并点击添加。类型选择文本，变量名称输入PASSWORD，值就是你的密码了，随便输入什么，只要记得住即可。然后点击右上角的访问按钮（就是那个长得像分享按钮的），在url后面加上斜杠和你的密码（比如twilight-lake-cbba.wdtei8z.workers.dev /pangbai123@ )，打开之后，就可以通过代理工具来使用了。

可惜的是，.dev域名几乎给中国墙完了。所以你需要拥有一个自己的域名来避免被墙。点击设置，找到域和路由，点击添加，输入你在Cloudflare的域名就好了。

如果说你没有将域名绑定在Cloudflare，或者没有域名，请继续往下看。

获取免费域名

在注册之前，我们要准备下面这些东西：

  • 海外身份信息
  • KYC验证文件
  • 真实的电子邮箱

前两者，除了老外，我们都是没有的，但是没关系。海外身份信息我们可以使用虚拟的，点击这里来生成虚假身份信息。

然后我们来到永久免费的二级域名注册网站——nic.us.kg。进入网站后，你会看到以下画面：

在带有Domain name的输入框中，输入你想要的域名（比如pangbai1234），点一下Check Availability，会跳转至登录页面，点底下的蓝字Sign up，来到注册页面。

这里的Username写一个你好记住的（也就是账号id），然后回到刚才生成虚假信息的网页，将圆形头像下面的名字粘贴到上方Legal full name处。E-Mail填写你的真实电子邮箱。Phone一栏将刚才网页的Telephone或者Mobile复制粘贴并删除 - ，前面再加上 +1-（比如666-777-1234，粘贴过去应改为+1-6667771234）然后填写Full Address，先将虚拟信息网页中的Street那部分粘贴（比如123 Neuport Lane）然后添加一个英文的逗号，再将City, State, Zip粘贴，（比如Gri, Geo (GE) , 11451）括号中遮挡部分要删除。

输入完成后，填写你的注册密码并验证人机身份即可。然后进入邮箱，验证一下电子邮件，注册结束！

接下来登录一下账号，会发现要求验证KYC，我们点击选择方法，再点击上传KYC文件，然后点击继续默认服务器，在下图第二个输入框中输入I’m a student, and I’m learning about website construction. So I must need a domain name.，第三个框中输入Student ID。接下来就要处理KYC了。

KYC只需要上传第一个即可，那个带有optional的是可选项的意思。点击这里，进入KYC生成网站。

Full Name一栏填写你刚才在注册免费域名的账号时所用到的名称。（如果忘记了，回到KYC网页，第一个框就是）

ID Number一栏可以随便填写一串数字。（比如23442123）

Institution Name一栏填写Baylor University。

Logo URL of the Institution一栏填写下面这个：

1 https://s.rmimg.com/2024/05/29/3a7b5b993d8aede065bfbd242364a3ca.jpg

Address of the Institution一栏填写Waco, TX, 76798。

Contact Number of the Institution一栏填写02547101011。

Template Choice默认即可。（默认为：学生卡带专业）

忙活了半天，终于填好了。点击Generate，生成虚假的KYC，将生成出来的图片截图保存，再上传至KYC Document(Front)中即可。再点击底部蓝色按钮，验证成功。

接下来回到nic.us.kg，登录你刚才注册的账号，你会看到这样的界面：

点击顶部的Register按钮，输入一个你想要的名字（比如pangbai1234），点一下Check，如果出现绿色框，说明该域名没被注册使用，如果是红色，那么换一个名字即可。

⚠️注意⚠️

域名注册成功时，你会注意到Price那一栏有这样的字：Always free, with a default registration period of one year (365 days). Renewal is available for the next year when there are fewer than 180 days remaining until expiry.翻译成中文是：永久免费，默认注册期限为一年（365 天）。距离到期时间不足 180 天时，可以续订下一年度。

由此可见，域名应该是需要手动续费的，不过续费也是免费。

往下看，会发现 Name Server 1 和 Name Serve 2 是必填项，现在，我们就要托管域名至Cloudflare上了。

托管域名至Cloudflare

回到Cloudflare，点击左侧的账户主页，再点击域，将你的域名粘贴上去（比如pangbai1234.us.kg），然后点击继续，再点击下方的Free，点击继续，然后往下滑，点击继续前往激活，跳出添加DNS记录弹窗，点击确认即可。

  • 然后往下滑，找到将您当前的名称服务器替换为 Cloudflare 名称服务器，将给出的Name Servers依次复制，然后来到你的域名服务商，将Name Server改为你刚才在Cloudflare中复制的即可。（由于域名服务商过多，所以更改方法自行搜索）
  • 或者回到us.kg，在你刚才免费注册域名的 Name Server1 和 Name Serve 2 中粘贴Cloudflare复制下来的内容，然后点击下方的Register，出现Thanks就是注册完成了。

回到Cloudflare，点击继续，等待域名解析生效，大概几分钟就行了。刷新网页，当顶部橙色的待处理的名称服务器更新变成绿色的活动，那么就是托管成功了。

将Workers中的域名替换

现在，我们已经托管了自己的域名。点击左上角的Cloudflare图标回到主页，进入你刚才部署的Workers的设置页面，点击添加，点击自定义域，再输入你刚才托管的域名即可（比如我的是pangbai1234.us.kg）。

最后，只要访问dev域名或者你刚才添加的域名，在后面跟上密码（比如这两个：cbba.wdtei8z.workers.dev /pangbai123@ ；pangbai1234.us.kg /pangbai123@ ）就可以成功进入你所搭建的梯子页面了。

通过代理工具来实现佛跳墙

这里可以使用v2rayN或者clash-verge（原版clash不支持vless）。软件请自行寻找，我这里不提供。

以v2rayN为例，进入软件后，点击蓝色底的加号，别名随便取一个，可选地址输入你刚才的网址以及密码（比如pangbai1234.us.kg/pangbai123@）点击确定，就出现节点了。Ctrl+A全选，再按下Ctrl+R来测定延迟，延迟越低越流畅。找到延迟低的后，点击该ip，按下enter键选中，再将下方清除系统代理改为自动配置系统代理，随后访问外国网站，会发现访问成功，那么就是搭建完成！
